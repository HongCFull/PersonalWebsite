Savior Zone

ShellFox is my first "hello world" game development project that made with my college friend Arthur It was started during the spring semester in my 3rd year of study at HKUST for the requirement of the course Comp4451
In which, I worked on the game logic programming while Arthur worked on the comic story creation and the shellfox model.

ShellFox is a single-player third person shooter (tps) sandbox game that borrows the idea of pokemon battle.
In ShellFox, the player acts as the shellfox for putrifying its friends who are contaminated by the dark sun!
Link to the Github page

What i have learnt in this project:

  • Basic Unity Workflow and C# scriptting
  • Procedural Terrain Generation using perlin noise
  • Simple procedural texture
  • AI agent decision making (State Machine)
  • Audio source Management
  • Basic particle system
  • Basic physics system
  • Basic animation system
  • Version control using Git
