Ancient Dungeon

Ancient Dungeon is my second unity game project. It is a third-person adventure game that the player has to control the character "Shirley" to clear a procedurally generated dungeon. It was started during the fall semester in my 4th (final) year of study at HKUST for the requirement of the course Comp4971C.

This is a solo project that aimed to train my software engineering skills like designing scalable architectures, writing manageable code using suitable design patterns, etc.
Link to the Github page

What i have learnt in this project:

  • Design Patterns : Singleton, State pattern, Factory pattern, Observer pattern and Strategy pattern
  • Procedural Dungeon Generation Algorithm
  • Animator (Animation Blending and Animation Layers)
  • AI agent decision making (Using the built in Animator Atate Machine)
  • Character Emotion Control (Using State Pattern)
  • Cinemachine (Camera Transition, Cut Scene)
  • Basic shader graph (Dissolve Effect)
  • UI elements
