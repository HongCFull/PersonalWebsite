Open World RPG

At the final studying year in HKUST, I worked with other 3 team members under the supervision of Prof. Horner for our final year project - An open world game created using Unreal Engine 4.
The playstyle is similar to most existing open world RPG

Game features:

  • Dynamic Sky System: Time-lapse and weather simulation
  • Inventory system: Storing collectable items in the player's inventory
  • Quest system: Player can take quest in the game
  • Multi-Weapons : The main character can use 9 different weapons
  • AI enemies: AI enemies with dynamic combat behaviors
  • BGM transition system: BGM transists between regions and combats

What i have learnt in this project :

  • Basic Unreal Engine workflows
  • Basic Unreal Engine architectures: (Actore, Pawn, Characters, Controller etc.)
  • BluePrints and Unreal C++
  • Unreal's Animation System (State machine etc.)
  • Unreal's AI system (Behavior Tree, BlackBoard, EQS etc.)
  • Unreal's Sound system
  • Unreal's Particle effect system
